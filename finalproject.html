Summary of my project:

The project for my AP Computer Science class was to create an object in java. I decided in the fall to create a project of an object and the object I choose was Football because I am really into watching and playing the sport. The traits of the football class are the following age, yearsnfl, rushingyards, catchingyards, name, lastname, jersey, fans, position, team, vetorrookie, and salary. The Football class implements myPlayer. The interface myPlayer only has one method and that method is success() which when is accessed will print the following statement “The potential for success is high”. Football has one player and Football has a child object which is myPlayer. In myPlayer the user can submit the name of the players agent and as well whether the player is good or bad. The average yards per game is given as a random. I tested the methods using a driver class for Football. I made it so the Fantasy Roster is provided and shows the cuts made on the Roster and all organizes it based on their salary from least to greatest.
