Summary of my project:

My spring project for my AP Computer Science class was to create an object in java. For my project, I decided to make an object of an airplane because I love to travel. The airplane has 8 attributes: airline, airplaneSize, capacity, full, cabin, engineOn, ticketPrice, and Weight. In addition, the airplane class implements an interface Fly. This interface contains one method- fly(). When fly() is accessed, it will print “The plane is in the air”. The airplane class can be limited to one specific cabin, and the price for each ticket is randomly determined. I created a child object called privateJet. This object allows the user to input a name as the owner of the jet and determine whether the jet has food or not. In the driver, I created 3 separate planes to test methods used in the Airplane class. I also let the user told the user that they don't have enough money to sit in their desired cabin so I let them decide if they wanted to list as standby. I tested the privateJet class with an object called Jet, and printed the fly method from the Fly() interface. In the next part of the program, I created a larger collection of airplanes, some that remain within an airport(array) and some that are lined up in a runway(ArrayList). Additionally, I created an array that contained all the weights of all the planes. I created a method that would retrieve the large planes from the array and turn the engines on and would also remove full airplanes from the arrayList during the holiday season. In addition, I used a quick sort method to reorganize all the planes by cheapest ticket price to most expensive. My method called getBiggestWeight would print out the weight of the heaviest airplane in my collection. From the driver, the entire collection of airplanes will be printed, as well as the heaviest plane in the collection.
